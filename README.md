# WPF.NotepadMinus
 A notepad++ replacement ( In progress)
 
 一个正在开发中的notepad++替代.
 
 Based on WPF,MVVM toolkit and AvalonEdit

## How to build
使用 Visual Studio 2019或更新版本打开解决方案，选择还原nuget包，之后进行编译运行。
